Chess Helper AI

Real-time chess analysis tool with neural network-powered move suggestions, position evaluation, and audio coaching. Deployed as a Chrome extension with FastAPI backend.

95% Accuracy
50ms Latency
10K+ Users

Technical Overview

Built using Stockfish engine integration, real-time board state analysis, and custom neural networks for position evaluation. The system includes:

  • Custom CNN for board position recognition
  • LSTM model for move sequence prediction
  • Real-time analysis engine with < 50ms latency
  • Audio synthesis for coaching feedback

Model Explainability

Used SHAP values to explain move recommendations and Grad-CAM for visualizing board attention maps. This ensures transparency in AI suggestions.
